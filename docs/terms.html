欢迎使用「问得好」Chrome 插件及相关 API 服务。使用本服务即表示您同意以下条款。

1. 服务说明

本服务是 AI 工具上的提问增强层，帮助用户发现缺失信息、优化提问质量。本服务不是独立的 AI 聊天产品，也不替代各 AI 平台的账号与服务条款。

2. 使用条件

  • 您应合法、合规使用本服务，不得用于违法、侵权、骚扰、欺诈等用途；
  • 您需自行保管 API Key，不得转借或公开分享；
  • 我们可能对免费用户设置每日调用次数上限；
  • 公开试用采用内置 API Key，并按 IP 地址限制每日调用次数（analyze 与 complete 各计 1 次）；
  • 全站每日总调用量达到上限时，服务可能暂时不可用，次日重置；
  • 您上传的问题内容需遵守各 AI 平台的使用规范。
