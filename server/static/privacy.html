问得好 隐私政策

更新日期：2026年6月

「问得好」（以下简称「本服务」）是一款 Chrome 浏览器插件，帮助用户在 ChatGPT、DeepSeek 等 AI 工具上优化提问。我们重视您的隐私，请仔细阅读本政策。

1. 我们收集的信息

  • 您输入的问题文本：用于分析问题完整度并生成增强建议。
  • 对话上下文：从当前 AI 页面提取的已发送消息，用于理解追问语境（不包含未发送的草稿以外的页面其他数据）。
  • 使用行为数据：如点击优化、分析成功/失败、耗时等，用于改进产品。
  • 匿名标识：插件生成的 clientId（随机 UUID），不含姓名、手机号等实名信息。
  • API Key：用于鉴权，存储在您的浏览器本地，不上传到除本服务后端以外的第三方。

2. 信息的使用目的

  • 提供问题分析与补全功能；
  • 调用 DeepSeek 等大语言模型 API 完成分析（您的提问内容会发送至模型服务商）；
  • 统计服务质量、控制调用频率、防止滥用；
  • 排查故障与改进产品体验。

3. 第三方服务

本服务使用 DeepSeek API 处理您的提问内容。DeepSeek 有其独立的隐私政策，请参阅 DeepSeek 官网。

4. 数据存储与安全

  • 使用数据存储在我们控制的服务器数据库中；
  • API 通信采用 HTTPS 加密；
  • 我们不会出售您的个人数据。

5. 数据保留

使用日志默认保留 90 天，用于统计与故障排查，之后可匿名化或删除。

6. 您的权利

您可以通过卸载插件停止数据上报。如需删除与服务相关的数据，请联系：support@wenhaode.com。

如有隐私相关问题，请发送邮件至上述地址。
